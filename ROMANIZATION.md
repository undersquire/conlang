# Romanization Guide

If you havent already, please read over the [Phonology document](PHONOLOGY.md).

Since the IPA symbols for the vowels and consonants are just typical ASCII characters, we simply use them
for the romanization.

| Symbol | Vowel |
|--------|-------|
| i | i |
| e | e |
| a | a |
| o | o |
| u | u |

| Symbol | Consonant |
|--------|-----------|
| m | m |
| p/b | p/b |
| n | n |
| t/d | t/d |
| l | l |
| k/g | k/g |
| h | h |
